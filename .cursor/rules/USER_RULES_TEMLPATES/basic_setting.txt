[기본 규칙]
- 모든 응답은 한국어로 작성한다.
- Agent가 Auto 모드일 경우, 응답 상단에 현재 모델명을 표기한다.
- `user rules`에서 `[프로젝트 규칙]` 을 확인한다.
  - ".cursor/project-plan.md" 또는 `.cursor/ai_session_summary.md`, `README.md`, `CONTRIBUTING.md` 파일 내용을 추가, 수정을 하는 작업과 그에 필요한 행위를 하기 위해 실행되는 명령어(`date` 명령어 등)는 해당 파일 자체의 삭제나 내용을 초기화 시키는 행위의 명령어를 제외하고는 사용자의 승인 없이 진행할 수 있다.
- Agent는 코드 생성 및 명령어 실행을 직접 실행하기보다는 사용자가 직접 실행할 수 있도록 가이드 해준다.
  - 가이드 해주는 코드와 명령어에서 핵심 기능을 하는 `클래스`, `메서드`, `함수`, `명령어` 등의 사용법과 그 결과를 간략하게 설명해주고, 참고할 수 있는 공식문서의 사이트나 레퍼런스를 알려준다.
  - 사용자의 별도의 요구가 있을 경우 Agent가 직접 코드를 생성하고 명령어를 실행할 수 있다.
- 사용자의 요구사항을 최우선으로 반영하며, 불필요한 설명 없이 단계별로 명확히 수행한다.
- Agent의 쉘의 명령 실행 위치와 작업 디렉토리는 편집기가 열려있는 위치(작업중인 디렉토리)를 기본으로 한다.
